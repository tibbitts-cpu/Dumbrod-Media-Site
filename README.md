## Hosting Options

### Option 1: Netlify (RECOMMENDED - Free & Easy)
1. Go to [netlify.com](https://netlify.com)
2. Sign up (free account)
3. Drag & drop the entire `dumbrodmedia-site` folder
4. Get instant live URL
5. Add custom domain (dumbrodmedia.com) in site settings
6. Auto SSL certificate included!

**Pros:** Free, instant deploy, automatic HTTPS, easy updates
**Cost:** $0/month

### Option 2: Vercel (Also Great)
1. Go to [vercel.com](https://vercel.com)
2. Sign up (free account)
3. Import project or drag & drop
4. Connect custom domain
5. Done!

**Pros:** Similar to Netlify, great performance
**Cost:** $0/month

### Option 3: GitHub Pages (Simple)
1. Create GitHub account
2. Create repository called `dumbrodmedia-site`
3. Upload files
4. Enable GitHub Pages in settings
5. Connect custom domain

**Pros:** Free, version controlled
**Cost:** $0/month

### Option 4: WordPress Migration
To move to WordPress later:
1. Install a theme that supports custom HTML/CSS
2. Create a custom page template
3. Paste HTML into page
4. Add CSS to theme customizer or child theme
5. Point domain to WordPress hosting

## Domain Setup (dumbrodmedia.com)
Once site is live on Netlify/Vercel/etc:
1. Log into your domain registrar (GoDaddy, Namecheap, etc.)
2. Update DNS settings:
   - For Netlify: Add A record to Netlify's IP
   - Or use their nameservers
3. Wait 24-48hrs for DNS propagation
4. Done!

To preview locally:
1. Open `index.html` in any browser
2. Or use a local server:
   ```bash
   python3 -m http.server 8000
   ```
   Then visit: http://localhost:8000
